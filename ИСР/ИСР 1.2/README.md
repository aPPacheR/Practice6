# Задание
1. Исследовать структуру официального сайта кафедры информационных технологий и электронного обучения (ИТиЭО) по адресу: https://ict.herzen.spb.ru и составить структуру сайта (дерево страниц сайта). 
2. Проанализировать 1-3 сайтов кафедр (и/или факультетов, выбрать наиболее качественные) других вузов (например, ВШЭ, МГТУ им. Н. Э. Баумана) и составить структуры их сайтов. Провести сравнительный анализ проанализированных ресурсов и описать какие разделы отсутствуют на сайте кафедры ИТиЭО и вы бы рекомендовали добавить.

3. Составить список сервисов (функций), которые вам, как студентам, были бы полезны и актуальны для использования на сайте кафедры. Составьте таблицу со столбцами ("Сервис/Функция, которая была бы актуальна на сайте кафедры", Инструмент реализации функционала", "Комментарий с аргументацией"). Опишите 5-10 необходимых, с вашей точки зрения, функций. Аргументируйте, по возможно лаконично, свой ответ. 



### Примечание
Дерево страниц сайта создать с использованием одного из онлайн-сервисов, в отчете предоставить изображение и указать какой сервис использовался.
