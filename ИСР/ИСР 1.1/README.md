# Задание
1. Провести инсталляцию программного обеспечения.

    На основе материала, инсталлировать необходимое ПО. В отчете поэтапно продемонстрировать процесс установки (в виде набора скриншотов) с ключевыми моментами установки.

2. Зарегистрироваться в сервисе figma и изучить дизайн альтернативной версии сайта кафедры.  Используя сеть Интернет, найти 2  различные темы оформления для сайта CMS Grav:

- тема №1 должна быть максимально похожа на разработанный альтернативный дизайн версии сайта кафедры в Figma,

- тема №2 должна быть быть максимально адекватна специфике внешнего оформления сайта кафедры (или сайта для вуза и/или его подразделения: факультета, кафедры);

3. Инсталлировать темы в развернутый локально сайт на Grav. И создать 1 информационную страницу в каждой теме, где можно было бы наглядно оценить её дизайн (страницу выбрать из существующих на сайте сейчас, например, эта). 

### Примечание 

В качестве референсов для темы №2 рекомендуется найти 3-5 сайтов такого рода (сайт для вуза и/или его подразделения: факультета, кафедры) (это будет полезно для Задания 1.2. ИСР) и понять как такая тема может выглядеть, какие разделы в ней будут и т.д.
